# GOLDLANDS Presentation

Clean standalone Hebrew RTL real estate presentation app for GOLDLANDS sales meetings.

## Run locally

```bash
npm install
npm run dev
```

Open `http://127.0.0.1:5173/`.

## Build

```bash
npm run build
```

## Current scope

- Mock data only
- No Supabase
- No Fireberry
- No uploads
- No real auth
- No external API integrations
